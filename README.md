# Weather Dashboard

## Functionality
### When viewing current weather conditions for the city, the following is shown:
-----
#### City name
#### Date
#### An icon of weather conditions
#### Temperature
#### Humidity
#### Wind speed

<br />

### When viewing future weather conditions for the city, the following is shown:
-----
#### City name
#### Date
#### An icon of weather conditions
#### Temperature

<br />

### Technologies used:
-----
#### HTML
#### CSS
#### Javascript
